### Hi, I'm Eleazar 👋

I'm a **Software Engineer** focused on backend systems, distributed architectures, and developer tooling — with enough frontend in my toolbelt to ship full-stack features when the work calls for it.

I also teach **Languages & Compilers** at **UNLaM (Universidad Nacional de La Matanza)**, where I help students build parsers, interpreters, and a working compiler from scratch.

- Sharing what I learn — in the classroom and in code
